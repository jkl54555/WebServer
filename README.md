## 設定檔

![](/docs/images/env.png)  

於".env"進行設定

### PORT

連接端口  
以3000埠為例，設定方式為"PORT=3000"  
若無設定使用網路協定預設  
http使用80  
https使用430  

### INTERFACE

預設網路介面，  
以Wi-Fi為例，設定方式為"INTERFACE=Wi-Fi"  
若無設定運行程式時會要求選擇
